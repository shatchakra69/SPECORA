# SPECORA

**Spec your AI, your way. Gain your aura.**

SPECORA is an AI chat bot, I built for anyone who wants a smarter, more personal assistant than a one-size-fits-all chatbot. It ships with six focused modes (Chat, Homework, Research, Humanizer, Code, and Creative), so the AI adapts to whatever I'm working on. I can sign in, keep my chat history, and drop in images or PDFs for the AI to read and reason over. Replies come back in a natural, human tone, never robotic. It's fast, minimal, and works on any device.

**Check the live demo here: [specora.live](https://specora.live)**

## Features

- Six AI modes: Chat, Homework, Research, Humanizer, Code, Creative
- Email and Google sign-in
- Searchable, per-account chat history
- Image and PDF attachments the AI reads directly
- Humanized replies: natural tone, no robotic phrasing
- Markdown rendering: code blocks, lists, and tables
- Editable profile and settings (name, avatar, preferences)
- Emoji picker
- Fully responsive on mobile and desktop
- Per-user rate limiting

## Tools and Services

- **Authentication:** Supabase Auth (email/password and Google OAuth)
- **AI:** Anthropic Claude (Sonnet 4.6) via the official Anthropic SDK
- **Cloud storage:** Cloudinary (image and PDF uploads)
- **Database and accounts:** Supabase (PostgreSQL)
- **Chat history:** stored per account in the browser

## Tech Stack

- **Frontend:** React, Vite, Axios, Supabase JS, React Markdown
- **Backend:** Node.js, Express
- **Dev tools:** Git and GitHub, npm, Vite
- **Deployment:** Vercel (frontend), Render (backend)
